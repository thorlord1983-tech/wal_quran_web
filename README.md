# 🕌 Wal Qalaam Quran Academy

A modern, responsive website for **Wal Qalaam Quran Academy**, built using **HTML, CSS, and JavaScript**.

## 🌟 Features

- Responsive design
- Modern Islamic green and gold theme
- Hero section
- About section
- Teacher profile
- Quran courses
- Why Choose Us section
- Contact form
- Floating WhatsApp button
- Smooth scrolling animations
- Mobile-friendly navigation
- Vercel compatible

## 📚 Courses

- Noorani Qaida
- Quran Reading
- Tajweed
- Hifz-ul-Quran
- Quran Translation
- Tafsir
- Islamic Studies
- Daily Duas

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript

## 🚀 Deployment

This website is ready to deploy on **Vercel**.

1. Push the project to GitHub.
2. Import the repository into Vercel.
3. Click **Deploy**.
